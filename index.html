PROJECTS

Free Food Finder

This Android mobile application was developed in order to help students on campus find and publish clubs or events with free food. The project was carried out for 8 weeks along with four other team members. Our goal was to meet our customer's needs by iteratively designing and implementing the application using an agile development process. My goal for this project was to do UI/UX design as well as front end implementation. I used paper to sketch out designs during customer meetings, then used Sketch 3 to create mock wireframes for evaluation. Once the design was settled, myself and one other team member developed the front end of the application.

Locate Me

Locate Me is a personal project that is meant to be a tool for people send out their location quickly through SMS, Facebook, Twitter, or a link. This project was a way for me to practice developing using an agile process. I started with a design in mind, mocked the deisgn up using Sketch, implemented valuable features, evaluated them, then repeated to see the outcome for any necessary changes. Locate me was developed using Android Studio, the Facebook SDK for Android, as well as Twitter's Fabric.
